## 🎯 Niche Selection (Recommended)

### Primary Niche: **Contemporary Romance — Small Town Romance**
- Romance = 37% of ALL Kindle Unlimited page reads (Q1 2026)
- Small town romance: high demand, manageable competition, emotional/repeatable
- Readers are LOYAL — they binge series (3-7 books)
- Tropes: second chance, grumpy/sunshine, enemies to lovers, forbidden love

### Secondary Niche (Phase 2): **Personal Growth / Mental Wellness**
- Evergreen demand, high reader loyalty
- Sub-focus: burnout recovery, overthinking, boundary-setting
- Non-fiction = faster to produce, no complex plotting

### Why These Niches?
- Both have proven bestseller rank sweet spots (#1 ~500, #100 ~10,000)
- Series potential (romance) + standalone potential (self-help)
- Both perform well in Kindle Unlimited (page reads = passive income)

## 🏗️ The 3-Layer Execution Plan

### LAYER 1 — Foundation (Weeks 1-4)
**Goal: Publish first lead magnet, start collecting emails**

#### Week 1: Setup & Niche Lock
- [ ] Choose pen name (romantic-sounding, memorable)
- [ ] Set up KDP account at kdp.amazon.com
- [ ] Set up Author Central page
- [ ] Create BookFunnel account ($15/mo)
- [ ] Create MailerLite account (free tier)
- [ ] Research top 100 small town romance books on Amazon
- [ ] Identify 5-7 winning tropes from bestsellers
- [ ] Study cover styles, fonts, color patterns in the niche

#### Week 2: Lead Magnet Creation
- [ ] Alex Prime writes 10,000-word lead magnet novella
  - Genre: small town romance
  - Tropes: 2-3 from research (e.g., second chance + small town bakery)
  - Structure: complete story with satisfying ending
  - Quality: edited, proofread, reader-ready
- [ ] Order cover on Fiverr ($20-30) matching niche style
- [ ] Format for eBook (EPUB)

#### Week 3: Infrastructure
- [ ] Set up BookFunnel landing page for lead magnet
- [ ] Create MailerLite welcome email sequence (3 emails)
  - Email 1: Book delivery + thank you
  - Email 2: Author introduction + what's coming next
  - Email 3: Recommendation + engagement
- [ ] Set up pen name social media (optional: Instagram, TikTok)

#### Week 4: Launch Lead Magnet
- [ ] Upload lead magnet to BookFunnel
- [ ] Test download flow end-to-end
- [ ] Lead magnet goes live — collecting emails 24/7
- [ ] Begin networking with other romance authors for future swaps

**Layer 1 Budget: ~$50**

---

### LAYER 2 — First Book & Optimization (Weeks 5-10)
**Goal: Publish Book 1 on Amazon, grow list to 1,500 subs**

#### Weeks 5-7: Book 1 Production
- [ ] Alex Prime writes 30,000-word novel
  - Same niche/pen name as lead magnet
  - Connected world (potential series starter)
  - Tropes readers responded to in lead magnet feedback
- [ ] Kabundji reviews key chapters for quality check
- [ ] Full edit pass (consistency, pacing, voice)
- [ ] Order professional cover ($30-50) — must match top sellers
- [ ] Format for Kindle + paperback (KDP Print)

#### Weeks 8-9: Amazon Optimization
- [ ] Keyword research: Amazon auto-suggest "drop-down method"
  - 7 keyword strings, 50 chars each
  - Target actual reader search terms
- [ ] Category selection: 3 winnable categories
  - Find similar successful books → copy their categories
- [ ] Blurb writing: Hook → Conflict → Cliffhanger formula
- [ ] Title optimization: include 2-3 tropes in title/subtitle
- [ ] Cover test: show to 5 people for 3 seconds — can they identify genre?

#### Week 10: Pre-Launch
- [ ] Set up KDP listing (don't publish yet)
- [ ] Begin booking newsletter swaps on BookClicker
- [ ] Email list update: "Book 1 is coming!"
- [ ] Set launch date (coordinate with swap schedule)

**Layer 2 Budget: ~$50**

---

### LAYER 3 — Launch & Scale (Weeks 11-24)
**Goal: Launch Book 1, publish Books 2-5, build income**

#### Week 11-12: Book 1 Launch
- [ ] Execute launch strategy:
  - Price at $0.99 for first 3-5 days (velocity)
  - 4 newsletter swaps/day during launch week via BookClicker
  - Email blast to full list
  - Raise price to $2.99-$4.99 after initial push
- [ ] Enroll in Kindle Unlimited (KU) for page read royalties
- [ ] Monitor sales, adjust keywords/categories as needed

#### Weeks 13-24: Publishing Pipeline
- [ ] Book 2: 30,000 words (same series) — publish Month 4
- [ ] Book 3: 30,000 words (same series) — publish Month 5
- [ ] Book 4: 30,000 words (series or standalone) — publish Month 6
- [ ] Book 5: 30,000 words — publish Month 7
- [ ] Each book cross-promotes the others (back matter links)

#### Scaling Activities
- [ ] Build ARC (Advanced Reader Copy) team for reviews
- [ ] Create box set of Books 1-3 when available
- [ ] Start testing Amazon Ads (only after 3+ books)
- [ ] Expand to secondary niche (personal growth) if romance performs
- [ ] Consider audiobook versions (ACX/Findaway)
