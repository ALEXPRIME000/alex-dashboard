# SECTION 1: Research & Niche Selection

## 1.1 How to Find Profitable Niches on Amazon

### Step 1 — Start with Amazon's Own Search Engine
Amazon search auto-suggest is a goldmine. It shows exactly what buyers are searching for.

1. Go to [amazon.com](https://amazon.com) (or amazon.co.uk, amazon.de for other markets)
2. Type: `coloring book` and pause — note every auto-suggestion
3. Type: `coloring book for` — note completions (kids, adults, seniors, toddlers, girls, boys…)
4. Type specific themes: `easter coloring book`, `dinosaur coloring book`, `mandala coloring book`
5. Note: Amazon shows you what REAL buyers type — this is keyword gold

### Step 2 — Browse Amazon's Bestseller Charts
1. Go to: **Amazon > Books > Children's Books > Arts, Music & Photography > Drawing > Coloring Books**
2. Or search: `amazon bestsellers coloring books`
3. Direct URLs:
   - Kids coloring: `https://www.amazon.com/Best-Sellers-Books-Childrens-Coloring/zgbs/books/4047`
   - Activity books: `https://www.amazon.com/Best-Sellers-Books-Childrens-Activity/zgbs/books/4046`
4. Study the **Top 100** — note:
   - What themes dominate
   - Seasonal vs evergreen mix
   - Price points
   - Cover styles
   - Page counts shown in product details

### Step 3 — Search Volume Validation
Type your niche into Amazon and check the **number of results**:
- `"easter coloring book"` → check results count in top bar
- Under 2,000 results = low competition (ideal for beginners)
- 2,000–10,000 = moderate competition (winnable with good SEO)
- 10,000+ = high competition (need strong differentiation)

- **Halloween** (Sept–Oct): 6–8 weeks of peak sales

**Pro:** Lower competition, clear timing window

1. Start with **1 seasonal book** (Easter/Christmas) for fast learning
2. Simultaneously build **2–3 evergreen books**
3. The seasonal book teaches you the process; the evergreen books build long-term income

# SECTION 2: Content Planning

## 2.1 Page Count Strategy

### The Math:
- **Target: 40–50 coloring pages** (the actual drawings)
- Each coloring page MUST have a blank page behind it (prevents bleed-through when kids color)
- 40 coloring pages × 2 = 80 pages interior
- Add: Title page + Copyright page + Welcome/Intro page + back matter = +6–8 pages
- **Final target: 86–108 total pages**

### KDP Minimum:
- KDP requires minimum **24 pages** for paperback
- Under 100 pages = thinner spine (harder to read spine text)
- **Recommended: 86–108 pages** — looks substantial, spine is readable

### Page Count Impact on Royalties:
- More pages = higher printing cost = lower royalty
- 108-page book at $9.99 → ~$3.42 royalty (white paper)
- 80-page book at $9.99 → ~$3.65 royalty (white paper)
- Don't go over 130 pages unless you raise price to $10.99–$12.99

---

## 2.2 Page Dimensions

### Recommended: 8.5" × 11" (US Letter)
**Why:**
- Largest standard KDP trim size
- Maximizes coloring space
- What parents expect for coloring books
- Easy to print at home if needed

### Alternative sizes:
- **8.5" × 8.5"** — Square format, unique look, higher printing cost
- **6" × 9"** — Standard book size, NOT recommended for coloring books (too small)
- **8.27" × 11.69" (A4)** — UK/EU standard, works well

### Interior Dimensions (with bleed):
- Trim size: 8.5" × 11"
- With bleed (for cover only, interior usually no bleed): 8.75" × 11.25"
- Safe zone (keep important content): 8.25" × 10.75" (0.125" margin all sides)

## 2.3 Interior Layout Structure

### Exact Page Order:
```
Page 1:  Title Page
Page 2:  Copyright Page
Page 3:  Welcome/Introduction (optional, adds warmth)
Page 4:  BLANK (back of intro)
Pages 5–6:   Coloring Page 1 (right side) + Blank (left side is back)
             WAIT — correction: See pattern below
```

### Correct Pattern for KDP Interior:
In a physical book, pages are printed on both sides. KDP prints:
- **Odd pages = right-hand side (recto)**
- **Even pages = left-hand side (verso)**

**For a coloring book, the pattern is:**

```
Page 1:  Title Page (right hand)
Page 2:  Copyright (left hand — back of title)
Page 3:  Coloring Page #1 (right hand — full page illustration)
Page 4:  BLANK (left hand — back of coloring page #1, faces coloring page #2)
Page 5:  Coloring Page #2 (right hand)
Page 6:  BLANK (left hand — back of coloring page #2)
... repeat for all coloring pages ...
Page [N-3]: Coloring Page #40 (or last page)
Page [N-2]: BLANK
Page [N-1]: "More Books by [Author]" page
Page [N]:   BLANK (back cover side)
```

> **Critical Rule:** NEVER put two coloring pages back-to-back. Always insert a blank page between them. This prevents marker/crayon from bleeding through and ruining the next image.

## 2.4 Style Consistency Rules

These rules are non-negotiable for professional quality:

### Line Weight:
- Choose ONE line weight and use it throughout: recommended **2–3pt stroke weight** for kids books
- Thinner lines (1pt) for adult coloring books (more detail)
- Thicker lines (3–4pt) for toddler books (easier to stay inside lines)
- **Never mix thin and thick line weights in the same book**

### Art Style:
- Pick ONE style: Cute/chibi, realistic, cartoon, geometric, whimsical
- Every image must look like it came from the same artist
- When using AI, use the SAME prompt structure and style keywords for every image
- Test with 3 images first — if they look inconsistent, refine your prompt before generating all 40+

### Complexity Level:
- **Toddlers (2–4):** Simple shapes, very thick outlines, large areas to color, max 5–8 elements per page
- **Kids (5–10):** Medium complexity, moderate detail, 10–20 elements per page
- **Adults:** High detail, intricate patterns, zentangle elements, 30+ elements per page

### Visual Consistency Checklist:
- [ ] Same art style throughout
- [ ] Same line weight
- [ ] Same complexity level
- [ ] Same page margins (consistent white border around each illustration)
- [ ] Same image placement (centered on page, consistent border)
- [ ] Consistent page orientation (all portrait or all landscape)

## 3.5 Converting AI Images to Clean Line Art

AI images often have slight grays, imperfect lines, or unwanted textures. Clean them up:

### Method A: Photoshop (Professional)
1. Open image in Photoshop
2. **Image > Mode > Grayscale** (discard color info)
3. **Image > Adjustments > Levels:** 
   - Push shadows slider right to ~150 (darkens lines)
   - Push highlights slider left to ~200 (whitens background)
4. **Image > Adjustments > Threshold:** Set to 128–160 (converts to pure B&W)
5. **Filter > Other > Maximum (1–2 px):** Slightly thickens lines (optional)
6. **File > Save As > PNG**

### Method B: GIMP (Free)
1. Open image in GIMP
2. **Image > Mode > Grayscale**
3. **Colors > Levels:** Adjust input levels
4. **Colors > Threshold:** Set to pure black and white
5. **Filters > Enhance > Unsharp Mask:** Sharpens lines
6. **File > Export As > PNG**

### Method C: Adobe Illustrator (Vector — Best Quality)
1. Open PNG in Illustrator
2. **Object > Image Trace > Make**
3. Set Mode to: Black and White
4. Threshold: 128
5. Paths: 50%, Corners: 75%, Noise: 25px
6. **Expand** the trace
7. **File > Export > Export As > PNG** at 300 DPI
8. *Advantage: Perfectly clean vector lines, scales perfectly*

### Method D: Vectorizer.ai or Adobe Express (Online, Free)
1. Upload PNG
2. Select "Line Art" conversion
3. Download clean SVG or high-res PNG
4. Works well for simple images

### What to Fix Manually:
- Broken line segments (use pen tool in Illustrator or brush in Photoshop)
- Areas where lines don't close (important for flood-fill coloring apps)
- Stray marks or specks (use eraser tool)
- Too-thin lines that will disappear in print (thicken with stroke in Illustrator)

# SECTION 4: Cover Design

## 4.1 KDP Cover Dimensions — The Math

### Critical Formula:
```
Total Cover Width = (Spine Width) + (Front Cover Width) + (Back Cover Width)
```

### Step 1 — Calculate Spine Width:
```
Spine Width (inches) = Page Count ÷ 2 × Paper Thickness

White paper (standard): Page count ÷ 444 inches per page = total inches
Cream paper: Page count ÷ 400 inches per page

FORMULA: Spine Width = (number of pages × 0.002252) inches [white paper]
FORMULA: Spine Width = (number of pages × 0.0025) inches [cream paper]
```

**Example:** 108-page book on white paper:
`108 × 0.002252 = 0.243216 inches ≈ 0.24 inches spine`

### Step 2 — Full Cover Template Dimensions:

For **8.5" × 11"** book with **108 pages** on **white paper**:

```
Front cover: 8.5" wide × 11" tall
Back cover: 8.5" wide × 11" tall
Spine: 0.24" wide × 11" tall
Bleed (each side): 0.125"

TOTAL WIDTH = 0.125 + 8.5 + 0.24 + 8.5 + 0.125 = 17.49"
TOTAL HEIGHT = 11" + 0.125 + 0.125 = 11.25"
```

**Full cover canvas size: 17.49" × 11.25" at 300 DPI = 5,247 × 3,375 pixels**

### KDP Cover Calculator:
Use the official KDP cover calculator: **kdp.amazon.com/en_US/cover-calculator**
Enter your: trim size, page count, interior type → it generates exact dimensions.

## 4.2 Cover Design Specifications

### Front Cover:
- Size: 8.5" + 0.125" bleed each side = 8.75" wide × 11.25" tall
- Safe zone: Keep all text and key artwork 0.25" inside trim lines
- Resolution: 300 DPI minimum, 300–600 DPI ideal
- Format: JPEG (KDP accepts JPEG for cover) or upload via template

### Back Cover:
- Same dimensions as front
- Include: Short description (optional), barcode placeholder (KDP adds automatically), ISBN area
- Keep lower-right 2" × 1.2" area clear for barcode

### Spine:
- Minimum readable spine: 0.25" width needed for readable text
- Under 0.25" spine: Leave spine blank (KDP won't print text)
- Over 0.25": Add book title vertically (rotated 90° counterclockwise — reads bottom to top when book on shelf)

**For Adults:**

- Dark or jewel-tone backgrounds work well

**Universal Rules:**
- Title must be readable at 100px thumbnail size
- Main image should be recognizable at 100px thumbnail
- Don't clutter — less is more
- Bright saturated colors outperform muted on Amazon thumbnails

## 4.5 Typography Guidelines

### Font Categories for Coloring Books:

**For Kids Books (5–10 years):**
- **Recommended:** Fredoka One, Nunito, Bubblegum Sans, Chewy
- Round, friendly, approachable
- Bold weight for title
- Avoid: Thin fonts, scripts, serif fonts

**For Toddler Books:**
- Extra bold, very simple letterforms
- Maximum 2 fonts (title + subtitle)
- Very large point size (title should be 60–90pt equivalent)

**For Adult Coloring Books:**
- **Recommended:** Playfair Display, Lora, Cinzel, EB Garamond, Cormorant Garamond
- Script accents: Great Vibes, Alex Brush, Sacramento

## 5.6 PDF Export Settings for KDP

### Critical Export Settings:
These are non-negotiable — wrong settings = rejected upload or poor print quality.

| Setting | Value |
|---------|-------|
| PDF Standard | PDF/X-1a:2001 (preferred by KDP) |
| Resolution | 300 DPI minimum |
| Color Mode | Grayscale (for B&W interior) |
| Bleed | 0.125" on all sides (if using full-bleed images) |
| Marks | Crop marks OFF (KDP doesn't want them) |
| Compression | Lossless or minimal |
| Font Embedding | ALL fonts must be embedded |
| Transparency | Flatten all transparency |

### How to Export from Different Tools:

**Canva:**
1. Download > PDF (Print)
2. Check "Crop marks and bleed" if using full-bleed images
3. Color profile: sRGB (Canva converts — acceptable)
4. Quality: High

**Adobe InDesign:**
1. File > Export > Adobe PDF (Print)
2. Preset: [PDF/X-1a:2001]
3. Marks and Bleeds: Set bleed to 0.125"
4. Output: Color Conversion = Convert to CMYK if color, Grayscale profile = Dot Gain 10%
5. Advanced: Font Downloading = Subset fonts below 100%

**Affinity Publisher:**
1. File > Export > PDF
2. Preset: PDF (for export) or PDF/X-1a
3. Enable Bleed, set to 0.125"
4. Colour Space: Greyscale (for B&W)
5. All Fonts Embedded: Yes

## 5.7 File Size Limits & Optimization

### KDP File Size Limits:
- **Interior PDF:** Maximum 650 MB
- **Cover PDF:** Maximum 40 MB

### If Your File is Too Large:
1. **Compress images before placing** — use TinyPNG or ImageOptim
2. **Convert all images to grayscale** before embedding (huge size reduction)
3. **Use JPEG instead of PNG** for interior images if file size is critical (set quality to 85–95%)
4. **In Acrobat:** File > Save As Other > Optimized PDF > Transparency and Images settings
5. **Target:** Under 200 MB for comfortable upload

### Optimization Checklist:
- [ ] All images are 300 DPI (not 600 — reduces size by 75%)
- [ ] Images are grayscale (not RGB)
- [ ] No embedded color profiles (can be stripped)
- [ ] Fonts subsetted (not full font sets)
- [ ] PDF/X-1a flattens transparency

### STEP 3: Book Content Page

**ISBN:**
- Option A: **"Get a free KDP ISBN"** — Amazon assigns a free ISBN
  - Downside: The publisher is listed as "Independently published"
  - Can only be used on Amazon KDP
- Option B: **Enter your own ISBN** — must purchase separately
  - US: Buy from Bowker (myidentifiers.com) — $125 for 1, $295 for 10
  - UK: Nielsen ($130 approx)
  - Advantage: You control the publisher name, can publish elsewhere
  - Recommendation: Free ISBN for beginners, own ISBN when scaling

**Publication Date:**
- Leave blank or set today's date (KDP will auto-populate)

**Print Options:**

*Interior & Paper Type:*
- Ink and Paper Type: **"Black & white interior with white paper"**
  - White paper = best for coloring (brighter, crisper lines)
  - Cream paper = slightly yellow tone, fine for text books, okay for coloring
- NEVER choose color interior unless you have a color interior (adds massive cost)

*Trim Size:*
- Select: **8.5" × 11"**
- This is the "US Letter" option in the dropdown
- If not listed, select "Custom" and enter 8.5 × 11

*Bleed Settings:*
- If your images go to the edge of the page: **"Bleed"**
- If your images have white margins: **"No bleed"**
- For standard coloring books with borders: "No bleed" is fine

*Paperback Cover Finish:*
- **Matte** — recommended for coloring books (professional, non-glare)
- Glossy — flashier, slightly cheaper to print, shows fingerprints

*Upload Interior File:*
- Upload your interior PDF (see Section 5.6 for export specs)
- KDP will process it (can take 5–30 minutes)
- If errors appear: Read the KDP Print Quality Check errors carefully and fix

*Upload Cover:*
- Upload your full cover PDF (with spine and back cover)
- OR use KDP Cover Creator (their tool — not recommended for professional books)

### STEP 4: Book Pricing Page

**Primary Marketplace:**
- Select Amazon.com (US) as primary

**Marketplace Prices:**

For a standard coloring book:

| Market | Recommended Price | Royalty Rate |
|--------|------------------|--------------|
| Amazon.com | $9.99 | 60% of list price minus printing cost |
| Amazon.co.uk | £8.99 | 60% minus printing |
| Amazon.de | €9.99 | 60% minus printing |
| Amazon.fr | €9.99 | 60% minus printing |

**Royalty Calculation for $9.99 at 8.5" × 11" (108 pages, white paper):**
```
List Price: $9.99
KDP royalty rate: 60%
60% × $9.99 = $5.994
Printing cost (108 pages, B&W, white, 8.5×11): ~$2.59
Your royalty: $5.994 − $2.59 = $3.40 per sale

Approximate: $3.40–$3.45 royalty per sale
```

**Expanded Distribution:**
- Enable this! Allows your book to be sold through libraries, bookstores, etc.
- Reduced royalty (40% instead of 60%) but wider reach
- Doesn't affect Amazon sales or royalties

# APPENDIX: Quick Reference

## Royalty Calculator Formula:
```
Royalty = (List Price × 0.60) − Printing Cost

Printing Cost (B&W, white paper, 8.5×11):
= $0.85 + ($0.012 × page count)
Example for 108 pages: $0.85 + ($0.012 × 108) = $0.85 + $1.296 = $2.15

Actual royalty at $9.99 / 108 pages:
= ($9.99 × 0.60) − $2.15
= $5.994 − $2.15
= $3.84 per sale

Note: KDP updates printing costs periodically. Always verify in KDP pricing tool.
```
