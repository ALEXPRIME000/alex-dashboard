## 🎨 Cross-Design Color Strategy

**High-priority color variants to always create:**
1. **Dark base** (Black/Navy/Charcoal) — highest seller for adults
2. **Light base** (White/Cream/Light Blue) — best for pastel/awareness niches
3. **Category color** (Autism Blue, Gold for Prom, School colors for AP)

**Rule:** Never launch with only 1 color. Minimum 3 variants per design.

## 🔑 Key Rules Recap

| Rule | Why It Matters |
|------|----------------|
| Text in design = text in title | Amazon algorithm match = organic ranking |
| No copyrighted fonts | Merch by Amazon TOS = account suspension risk |
| Simple bold text outperforms complex art | Thumbnail readability at 150px thumbnail size |
| Year-date every time-sensitive design | Creates urgency + keepsake value |
| "In My [X] Era" is trending NOW | Search volume is active — ride the wave |
| No puzzle pieces for autism | Community prefers infinity symbol — avoid negative reviews |
| Dark shirts for humor/adult designs | Converts better in those niches |
| Pastel shirts for kids/awareness | Category aesthetic expectation |
