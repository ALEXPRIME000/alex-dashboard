## BOOKFUNNEL SETTINGS CHECKLIST

When setting up the BookFunnel page, configure:

- [ ] **Landing page background:** Warm/cozy aesthetic — consider using a bakery image or illustrated small town scene
- [ ] **Cover image:** Your novella cover (front-facing, good resolution)
- [ ] **MailerLite integration:** Connect via BookFunnel → Integrations → MailerLite
- [ ] **Welcome group/tag:** Apply `novella-download` tag on signup
- [ ] **Confirmation email:** Use double opt-in OR single opt-in (single opt-in = higher completion, double = cleaner list — your call)
- [ ] **Download formats:** EPUB + MOBI + PDF (cover all devices)
- [ ] **Thank you page:** Use the copy above
- [ ] **Series/author branding:** Add author bio photo + short bio on landing page
