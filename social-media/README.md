## Design System

All assets feature:
- **Glass-morphism** — Frosted glass panels with subtle borders
- **Gradient backgrounds** — Rich dark-to-mid tone gradients
- **Glow accents** — Soft radial color glows for depth
- **Grid/dot patterns** — Subtle geometric overlays
- **Bold typography** — DejaVu Sans Bold + ExtraLight pairing
- **Brand-consistent colors** — Each company's palette applied consistently

## Platforms Covered

- LinkedIn (profile, cover, posts)
- Instagram (profile, posts, stories, reels)
- TikTok (profile, stories/covers)
- YouTube (profile, channel banner)
- Facebook (profile, cover, posts, stories)
