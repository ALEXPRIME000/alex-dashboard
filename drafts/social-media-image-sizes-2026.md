## ðŸŽ¨ Color Space & Export Settings

### Color Space Recommendations

**Universal Rule: Use sRGB IEC61966-2.1 for all web/social media**

- **Working Color Space:** Adobe RGB or ProPhoto RGB (editing)
- **Export Color Space:** **sRGB** (mandatory for web)
- **Why:** Browsers and social platforms assume sRGB. Other profiles will display incorrectly.

#### Photoshop Export
1. File â†’ Export â†’ Export As (or Save for Web - Legacy)
2. Check "Convert to sRGB"
3. Format: JPG or PNG
4. Quality: 80-100% (JPG), PNG-24 (PNG)
5. Uncheck "Embed Color Profile" for smaller file size (sRGB assumed)

#### Figma Export
- Figma exports in sRGB automatically
- Settings: PNG (lossless) or JPG (80-100% quality)

#### Lightroom Classic
- Export with Color Space: sRGB
- Quality: 90-100 (JPG)
- Resize to Fit: Long Edge (set pixel dimensions)

### Compression Best Practices

| Platform | Format | Quality | Notes |
|----------|--------|---------|-------|
| LinkedIn | JPG | 85-95% | Moderate compression acceptable |
| Instagram | JPG | 100% | Upload highest quality (IG compresses) |
| Facebook | JPG | 85-90% | FB heavily compresses anyway |
| YouTube | PNG | 100% | Thumbnails: max quality |
| X (Twitter) | JPG/PNG | 90-100% | Compresses on upload |
| TikTok | PNG | 100% | Cover images: lossless preferred |
| Pinterest | JPG | 85-95% | Balance quality/file size |

### File Format Guide

- **JPG:** Photos, complex images, gradients (use for most posts)
- **PNG:** Graphics with transparency, logos, text-heavy images, sharp edges
- **GIF:** Animations (max 15 MB on Twitter), moving images
- **WebP:** Modern format (supported on Twitter/Facebook 2026+), smaller than JPG
- **MP4:** Videos (universal support)
- **MOV:** Videos (alternative, Apple ecosystem)

- **Figma:** Layout Grid (Grid: 10px, Columns for safe zones)

- Stories (all platforms): 250px top/bottom, 100px sides
- YouTube Banner: 1235Ã—338 px center rectangle (critical content)
- LinkedIn Banner: Avoid bottom-left 268Ã—396 px
- Facebook Cover: Avoid bottom-left 170Ã—170 px

#### 4. Typography Best Practices
- **Minimum font size:** 24px (stories), 36px (posts)
- **Headline:** Bold, sans-serif, high contrast
- **Body:** Readable sans-serif (Open Sans, Roboto, Poppins)
- **Line height:** 1.2-1.4 for headlines, 1.5 for body
- **Alignment:** Left or center (avoid justified on social)
- **Hierarchy:** Clear size/weight difference between headline/body

#### 5. Grid & Alignment
- Use 8px or 10px grid system
- Align elements consistently
- Balance negative space
- Rule of thirds for focal points

- [ ] Font pairing (2-3 fonts max: headline + body + accent)
